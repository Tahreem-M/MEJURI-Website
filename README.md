# Mejuri Website
A modern, interactive front-end website named **Mejuri**, designed to showcase products and provide a seamless shopping experience. This project focuses on creating an attractive and user-friendly interface with functional features like login, signup, product cart, and dynamic displays.

## Features

###  Product Showcase
- Sections for **Necklaces**, **Bracelets**, **Rings**, and **New Arrivals**.
- **Carousel sliders** for product previews and customer reviews.
- Interactive product selection for an engaging user experience.

###  User Authentication
- **Signup** and **Login** options using **local storage** to save user credentials.
- Personalized shopping experience based on logged-in users.

###  Shopping Cart
- Users can **add products to the cart**.
- Cart **calculates total prices automatically**.
- Dynamic updates as users add or remove items.

### Design & UI
- Modern, responsive design that looks attractive on all devices.
- Smooth transitions and sliders for better interactivity.
- Organized layout for easy navigation.
